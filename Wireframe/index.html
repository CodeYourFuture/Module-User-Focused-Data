What is git?

Git is a version control software that allows developers to create and manage different versions of a project.

Why do developers need git?

Developers use git to handle large and small projects more efficiently track changes in the source code

What is branch in git?

Branches are for experimentation to try out difference styles with different branch names. E.g. try-purple or try-blue to experiment different colour themes. They may share some commonality in history and content but may vary in others.
